Les 07 - Personal HTML & CSS One pager

Wat heb ik allemaal geleerd?

Aan het begin wist ik helemaal niks van coderen. Ik heb geleerd dat HTML voor de structuur is. Ik heb geleerd om in HTML titels, tekst, foto's en een footer te maken. Ook heb ik bepaalde tags geleerd zoals Div om groepen te maken. Ik ging op het begin wat de mist in omdat ik heel chaotisch was met mijn bestanden en dus mijn pagina niet altijd deed wat ik wou. Hierdoor heb ik geleerd dat het heel erg belangrijk is om alles goed gestructueerd te houden.

Tijdens de eerste lessen waar wij een pagina moesten maken van een recept was ik al wat bezig met CSS. Nog niet goed wetende waar ik mee bezig was, was het mij wel gelukt om positie's, kleuren en fonts te bewerken in mijn pagina. Later in de lessen kwam ik er dus achter dat dit CSS is, en dat het niet de bedoeling is dat ik deze samen in mijn HTML bestand doe. Ik heb geleerd om HTML met CSS te linken en Div-ID's te maken.

Wat zijn mijn actiepunten?

  • Positions
    • Ik zou graag meer te weten willen komen over positions zoals absolute en relative omdat ik daar nog wel veel mee aan het puzzelen ben.
  • Javascript
    • Ook zou ik de pagina interactief willen maken d.m.v. Javascript. Ik wil ook bewegende elementen in een pagina verwerken. Ik moet allemaal nog goed uitzoeken of dit met html, css of javascript kan.
  • Webdesign
    • Ook zou ik wat meer kennis willen gaan op doen over web design/layouts.
  • Menu
    • Ook zou ik willen leren om een menu te maken en meerdere pagina's.
